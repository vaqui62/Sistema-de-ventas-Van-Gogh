# Tienda de Ropa Van Gogh

## Descripción del Proyecto

Tienda de Ropa Van Gogh es una plataforma diseñada para gestionar de manera eficiente una tienda de ropa mediante una base de datos centralizada.

Su objetivo es facilitar la administración del negocio, permitiendo la interacción entre gerentes, empleados y clientes de forma intuitiva, rápida y segura.

El sistema busca mejorar la organización del inventario, optimizar las ventas y ofrecer una experiencia de usuario clara y fácil de usar.

---

## Objetivos

* Crear una base de datos organizada para la gestión de productos, empleados y ventas.
* Facilitar el acceso a la información según el rol del usuario.
* Mejorar la experiencia de compra de los clientes.
* Permitir a los gerentes controlar y administrar el sistema de forma eficiente.

---

## Tipos de Usuarios

### Gerentes

* Añadir, editar y eliminar empleados.
* Gestionar productos.
* Supervisar el inventario.
* Acceder a reportes del sistema.

### Empleados

* Administrar inventario.
* Actualizar productos.
* Gestionar ventas.

### Clientes

* Visualizar productos.
* Realizar compras.
* Navegar fácilmente.

---

## Funcionalidades Principales

* Gestión de inventario
* Administración de usuarios
* Sistema de compras
* Búsqueda y filtrado de productos
* Control de ventas
* Interfaz amigable

---

## Tecnologías Utilizadas

* Backend: Python
* Base de Datos: PostgreSQL
* Frontend: HTML, CSS, JavaScript
* Control de Versiones: Git y GitHub
